## üöÄ Quick Start

### Prerequisites
```bash
Node.js v18+
MongoDB 5.0+
npm or yarn
```

### Installation

**1. Clone the repository**
```bash
git clone https://github.com/DocHarry22/coursecampass.git
cd coursecampass
```

**2. Backend Setup**
```bash
cd backend
npm install

# Install additional dependencies for advanced features
npm install compression helmet json2csv

# Create .env file
echo "MONGODB_URI=mongodb://localhost:27017/coursecompass
JWT_SECRET=your_jwt_secret_key_here
EMAIL_USER=your_email@gmail.com
EMAIL_PASS=your_app_password
PORT=5000
FRONTEND_URL=http://localhost:3000" > .env

# Seed database (optional)
node config/seedDatabase.js

# Create database indexes for performance
node scripts/createIndexes.js

# Start server
npm start
# or for development
npm run dev
```

**3. Frontend Setup**
```bash
cd ../react-admin
npm install

# Install charting library
npm install recharts

# Start development server
npm start
```

**4. Access the Application**
- **Frontend:** http://localhost:3000
- **Backend API:** http://localhost:5000/api
- **MongoDB:** mongodb://localhost:27017/coursecompass
